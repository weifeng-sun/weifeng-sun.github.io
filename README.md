# Weifeng Sun's Academic Homepage

This site is built with the [al-folio](https://github.com/alshedivat/al-folio) Jekyll theme.

## Edit content

- Homepage: `_pages/about.md`
- Publications: `_bibliography/papers.bib`
- Venue colors/abbreviations: `_data/venues.yml`
- Social/contact links: `_data/socials.yml`
- Profile photo: `assets/img/prof_pic.jpg`
- PDF files: `assets/pdf/papers/`

Blog, projects, repositories, and teaching pages are intentionally not included.

## Build locally

Use Ruby 3.3.x.

```bash
bundle install
bundle exec jekyll serve
```
